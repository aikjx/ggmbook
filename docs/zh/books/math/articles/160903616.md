python manage.py startapp promotion

- 拷贝模型

python manage.py makemigrations
python manage.py migrate

### 🧠 第二步：AI 大脑接入（核心灵魂）

- 建议微调：在 Prompt 最后加一句：“禁止使用‘最’、‘第一’、‘顶级’等《广告法》违禁词。”

## 全域数学半自动推广机器人 通俗白话版

咱们这个推广机器人，说白了就是商家的免费智能推广小助手，跟跑网约车一个道理，好理解又不用费脑子：

- 

机器人干累活

你不用自己绞尽脑汁写文案、做海报，把店铺、产品信息填进去，机器人立马帮你写好多条适合发抖音、小红书、朋友圈的推广话，还能自动做出带专属推广码的好看海报，全程不用你动手写一个字。

- 

你只干轻松活

机器人把文案、海报做好，你看一眼没问题，点一下发布按钮就行，不用一直盯着、不用刷屏群发，既省心，也不会被平台判定违规封号，安全又省事。

- 

赚钱跟网约车一样简单

- 网约车：系统派单，你接单送人，跑完单系统自动给你算钱、分钱

- 这个推广：机器人给你“派推广单”，你点一下发布，有人通过你的推广下单买东西，系统自动按规矩给你算提成、打钱，不用自己对账、要钱，躺赚推广佣金。

- 对商家/普通人都友好

- 商家：不用雇推广、不用花广告费，机器人免费帮你做推广，卖出去货大部分钱归自己

- 推广的人：不用囤货、不用懂营销，点一下发布就能赚提成，越推广赚越多

简单说：机器人替你动脑做素材，你只动手点一下，成交了钱自动到账，零难度、零风险，人人都能用、人人都能赚。
