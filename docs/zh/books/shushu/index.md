## 易经数理

《易经》不仅仅是占卜之书，更是一套完整的数理体系。

$$\text{太极生两仪}, \text{两仪生四象}, \text{四象生八卦}$$

## 道德经解析

道生一，一生二，二生三，三生万物。

$$道 = 0, 一 = 1, 二 = 2, 三 = 3$$
