# 全域数学vs传统数学：人类文明进阶200讲

## 课程概述

这是一套从基础到进阶的数学课程，共200讲，帮助您从全新视角理解数学。

## 课程结构

### 第1-50讲：基础概念

- 第1讲：数字是如何生长的
- 第2讲：交换律的本质
- 第3讲：乘法的真正含义
- 第4讲：分数的本源
- 第5讲：小数的刻度
- 第6讲：三角形内角和
- 第7讲：圆形的奥秘

### 第51-100讲：初中数学

$$a^2 + b^2 = c^2$$

$$y = mx + b$$

### 第101-150讲：高等数学

$$\frac{dy}{dx} = \lim_{\Delta x \to 0} \frac{f(x+\Delta x) - f(x)}{\Delta x}$$

$$\int f(x) dx$$

### 第151-200讲：跨学科应用

将数学应用于物理、AI、哲学等领域。
