# 书籍总览

欢迎来到全域数学书籍体系！这里汇集了四大核心书籍：

## 四大核心书籍

### 🔢 全域数学
探索数学的终极统一理论，从底层公理到应用实践的完整体系。

[进入学习](/zh/books/math/)

### 🔍 哥德巴赫猜想
深入研究数论经典难题，探索质数分布的奥秘。

[进入学习](/zh/books/goldbach/)

### 📊 数术工坊
融合传统数术与现代数学，探索古老智慧的数学本质。

[进入学习](/zh/books/shushu/)

### 📚 文明进阶200讲
全域数学与传统数学的深度对比，探索人类文明的进阶之路。

[进入学习](/zh/books/course/)

### 📝 CSDN博文
精选技术博文，涵盖数学、计算机科学等领域。

[浏览博文](/zh/books/articles/)

## 两套并行分类体系

### 体系1：底层四维本源统一分类

| 分类层级 | 符号命名 | 核心定义 |
| ---- | ---- | ---- |
| 1.奇点底层层 | $0$ 层 | 一切理论第一公理、本源假设、体系起点 |
| 2.动态演化层 | $\sim$ 残差层 | 波动、演化、误差、相变、周期、生命运动 |
| 3.结构约束层 | $\kappa$ 384爻网格层 | 固定规则、定律、结构、制度、逻辑框架 |
| 4.观测表象层 | Proj投影层 | 人类观测结果、现象、数据、感知、符号 |
